# Tic Tac Toe 🎮

A modern Tic Tac Toe game built using Python and Tkinter.

## Features
- AI opponent (Easy, Medium, Hard)
- Unbeatable AI (Minimax)
- Player vs Player mode
- Modern UI

## How to Run
python main.py
